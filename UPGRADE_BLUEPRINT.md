Store "Gems" earned through play in a "Locked Bank." User pays $0.99 to break it.

```javascript
const PIGGY_CAPACITY = 500;
function addToPiggyBank(amount) {
    let current = localStorage.getItem('piggy_count') || 0;
    current = Math.min(PIGGY_CAPACITY, parseInt(current) + amount);
    localStorage.setItem('piggy_count', current);
    
    if (current >= PIGGY_CAPACITY) {
        showPiggyOfferModal();
    }
}
```

## 4. Game Design & Retention Systems

### A. Daily Streak System
```javascript
function updateStreak() {
    const lastPlay = localStorage.getItem('last_play_date');
    const today = new Date().toDateString();
    let streak = parseInt(localStorage.getItem('streak_count') || 0);

    if (lastPlay === today) return; // Already counted
    
    if (isYesterday(lastPlay)) {
        streak++;
    } else {
        streak = 1;
    }
    
    localStorage.setItem('streak_count', streak);
    localStorage.setItem('last_play_date', today);
    return streak;
}
```
